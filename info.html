Objetivo del sitio

EnglishPro fue diseñado como una propuesta de sitio web informativo e interactivo para mostrar cursos online de inglés de forma clara, moderna y funcional. Se aplicaron etiquetas semánticas, estilos externos y scripts para mejorar la experiencia de usuario.

Tecnologías utilizadas

  • HTML5 para la estructura del contenido.
  • CSS3 para el diseño visual y la versión responsive.
  • JavaScript para menú móvil, validación y componentes interactivos.
  • Normalización de estilos para mayor compatibilidad.

Créditos del proyecto

Diseño

Se eligió una interfaz limpia, con colores modernos, tarjetas y espacios amplios para facilitar la lectura.

Estructura

El sitio cuenta con una página principal, una sección de precios y una sección de información, conectadas por una barra de navegación fija.

Interactividad

Incluye tema claro/oscuro, slideshow, acordeón y validación de formulario en el cliente.

Recomendación de uso

Este proyecto puede presentarse como ejemplo académico, portafolio o base para un sitio real de cursos. Solo necesitas cambiar textos, imágenes y datos de contacto para adaptarlo a una marca propia.
